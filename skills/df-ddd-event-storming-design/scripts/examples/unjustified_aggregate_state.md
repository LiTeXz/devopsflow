# 聚合设计

## 货架聚合

### 聚合属性

| 属性 | 说明 |
| --- | --- |
| 货架身份 | 主键 |
| 货架名称 | 页面展示 |
| 来源类型 | 投影需要 |
| 当前状态 | 持久化记录需要 |

### 规则

- 货架名称展示在列表中。

# 读模型
- 货架列表读取货架名称、来源类型和当前状态。

# 完整性检查
- 货架聚合属性来自持久化记录和读模型展示。
